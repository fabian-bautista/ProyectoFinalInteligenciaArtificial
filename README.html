Proyecto Final de Inteligencia Artificial

En este repositorio se encuentra el proyecto final de la clase de inteligencia artificial. Se realizó a partir de un dataset de Kaggle que relaciona ciertas características con enfermedades cardíacas para crear un modelo de predicción utilizando regresión logística y random forest.

Requisitos

Asegúrate de tener Anaconda instalado.

Instalación

  1. Clona el repositorio:
    2. git clone https://github.com/tuusuario/tuproyecto.git
cd ProyectoFinalInteligenciaArtificial
  2. Crea un entorno virtual con Anaconda:

    3. Para Windows:

    python -m venv myenv
.\myenv\Scripts\activate

    Para Linux/Mac:

    python3 -m venv myenv
source myenv/bin/activate
  3. Instala las dependencias:
    4. pip install -r requirements.txt
